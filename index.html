Project

Launch your own restaurant review site

Restaurant Review App

A simple, easy to use web application that offers restaurant reviews in the map area around the user's position. Restaurants are shown on the map based on their GPS coordinates. Restaurants that are currently visible on the map are displayed in list form on the side of the map. User will see the average reviews of each restaurant (ranging from 1 to 5 stars). A filter tool allows to display restaurants that have between X and Y stars. When user click on a restaurant, list of reviews and street view photo is shown. User can add new opinion about an existing restaurant, as well as add a new restaurant by right-clicking the map on the map.

Technologies

Restaurant Review App was created in React JS, HTML5, CSS3 and JavaScript (ES6). Application uses Google Maps API, Google Places API, Google Street View API, Google Reverse Geocoding API and Geolocation API. To read user manual or to see full project description, please see project repository on Github.

Project

Build a turn-based board game in JavaScript

Knight's Game

A turn-based board game in JavaScript. 2 players play each turn to compete. The map is randomly generated. On the map, a limited number of weapons is placed randomly and can be collected by players who pass through. For each turn, a player can move from 1 to 3 boxes before ending their turn. If a player passes over a box containing a weapon, they leave their current weapon on site and replace it with the new one. If players cross over adjacent squares, a battle begins. Each player attacks in turn. The damage depends on the player's weapon. The player can choose to attack or defend against the next shot. If the player chooses to defend themselves, they sustain 50% less damage than normal. As soon as the life points of a player falls to 0, the game is over.

Technologies

Knight's Game was created in HTML5, CSS3, JavaScript (ES5) and jQuery. To read more about rules I had to follow in the process of creating the game or to see full project description, please see project repository on Github.

Project

Create a quotation generator

Quote Generator

A simple, easy to use quotation generator. Randomly generates quotes by assembling sentence fragments. The quotes are generated one by one with each JavaScript call. Each quote is a combination of at least 3 sentence fragments (beginning, middle and end). User can select the number of quotes to generate (between 1 and 5), number of fragments to create each quote (between 3 and 5) and also choose in between two types of quotes to generate.

Technologies

Quote Generator was created in HTML5, CSS3, JavaScript (ES5) and Bootstrap 4. To read more about rules I had to follow in the process of creating quotation generator or to see full project description, please see project repository on Github.

Project

Map out a technical solution for the food delivery site Express Food

Express Food SQL Database

Relational database created in SQL for takeaway restaurant.

Database contains among others, the following tables :

  • Addresses
  • Clients
  • Clients addresses
  • Clients orders
  • Daily menus
  • Delivery persons
  • Menu items
  • Orders

Database contains 4 views. List of clients, List of delivery people, List of different daily menu items and Order history for each client.

Technologies

First I used UML (Unified Modeling Language) to create UML domain diagram. Then physical data model was created. To create database I used XAMPP and phpMyAdmin. To see full project description, please see project repository on Github.
